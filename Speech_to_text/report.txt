Для решения задачи преобразования речи в текст была использована библиотека transformers от Hugging Face, которая предоставляет доступ к различным предварительно обученным моделям, включая Wav2Vec2. Это модель на основе трансформеров, которая эффективно решает задачу распознавания речи.

Преимущества библиотеки:

Гибкость: Поддержка множества языков и моделей для решения различных задач в области обработки естественного языка и аудио.
Удобство использования: Простота интеграции и быстрота в развертывании моделей с предобученными весами.
2. Принцип работы модели
Модель Wav2Vec2 использует подход на основе трансформеров для обработки аудиофайлов. Принцип работы модели следующий:

Аудиофайл подается на вход модели.
Модель преобразует аудио в последовательность признаков.
После обработки модель предсказывает текст, который соответствует этим признакам.
Модель была выбрана по нескольким причинам:

Эффективность: Wav2Vec2 достигает высоких результатов на различных датасетах для распознавания речи.
Предобученность: Модель уже обучена на большом корпусе данных (на данных 960 часов речи), что позволяет использовать её без необходимости в дополнительном обучении.
3. Структура датасета
Модель facebook/wav2vec2-base-960h была обучена на большом датасете с аудиофайлами, содержащими записи речи на английском языке. Каждый файл в датасете представляет собой аудиофайл с расшифрованным текстом. Эти аудиофайлы имеют стандартную частоту дискретизации 16 кГц.

В нашем примере для тестирования модели был использован качественный аудиофайл в формате .wav с частотой дискретизации 16 кГц, что является оптимальным для работы модели.


Особенности реализации алгоритма и его эффективность
Загрузка модели и процессора: Для загрузки модели использовались классы AutoProcessor и AutoModelForCTC, что позволило легко загрузить предобученную модель и токенизатор для обработки аудио.
Обработка аудио: Для загрузки и обработки аудиофайлов использовалась библиотека librosa, которая позволила легко загружать файлы с нужной частотой дискретизации (16 кГц).
Распознавание речи: Модель преобразует аудио в текст с использованием метода декодирования, где применяется логит на основе вероятностей, что позволяет эффективно извлекать текст из аудиофайлов.
Эффективность модели: Модель показала хорошую производительность и адекватные результаты на качественном аудиофайле. Распознанный текст был точным и соответствовал содержанию аудио.
